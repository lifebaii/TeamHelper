# 全自动加入Team

解压插件

地址栏输入 edge://extensions/

点击开发者模式，加载加压缩的扩展

找到 ChatGPT Team Helper，点「详细信息」拉到最底下，把「在 InPrivate 中允许」开关打开

点击加入全部后，自动加入team，全部加入后点击 下载全部JSON
